## Available Google Dorks

- **site:** - Search within a specific site
- **filetype:** - Search for a file type
- **intitle:** - Words in the title
- **inurl:** - Words in the URL
- **intext:** - Words in the content
- **cache:** - Cached version
- **link:** - Pages with links to
- **related:** - Similar sites
- **info:** - Information about a URL
- **define:** - Definition of a term
